Execution commands for the models

syntax : python <file-name> <Classifier-name>


1. Linear Regression

            python main.py linearR

2. Decision tree Regression

            python main.py dtree

3. Random forest Regression

            python main.py rforest

4. Cataboost Regression

                 python main.py catboost

5. Support Vector Regression

                 python main.py svm
